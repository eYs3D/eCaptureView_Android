## Prerequisites 
Android 5.0 (21) ~ Android 10 (29)  

## How to Install ADB
https://developer.android.com/studio/command-line/adb

## Installing the eCapture apk to Android device
```sh 
adb install eDepthPreview_v1.0.0.0.apk
```
